Documentacion de API

Introduccion

En esta documentación le ayudaremos a familiarizarse con los recursos de la API de Estudiantes y Tareas, le muestra cómo realizar diferentes consultas, para que pueda aprovecharlas al máximo.

URL base:

http://localhost:3005/

La URL base contiene la pagina principal

API ESTUDIANTES

URL TODOS LOS ESTUDIANTES:

http://localhost:3005/api/estudiante

La URL contiene información sobre todos los recursos de API disponibles. Todas las solicitudes son GET solicitudes.
Todas las respuestas devolverán datos en formato json. Por cada Estudiante podrá ver :

  • Cedula
  • Nombre
  • Nota 1
  • Nota 2

URL SOLO UN ESTUDIANTE:

http://localhost:3005/api/estudiante/0706191558

La URL contiene información sobre un solo estudiante, dependiendo del numero de cedula que se ingrese, se mostrara la informacion del estudiante con la cedula ingresada.

BUSCAR POR NOMBRE:

http://localhost:3005/api/estudiante?nombre=Andy

Esta URL se encarga de buscar al los estudiantes por nombre.

URL PAGINACION:

http://localhost:3005/api/estudiante?page=0&size=2

La URL clasifica la informacion por paginas, se agregan dos atibutos al link: page (la pagina en la que se encuentra) y size (numero de elementos que se encuentra en esa pagina)

BUSCAR POR NOMBRE EN UNA PAGINA:

http://localhost:3005/api/estudiante?page=0&size=2&nombre=Angeles

Esta URL se encarga de buscar por nombre al estudiante en la pagina deseada

API TASK

URL TASKS:

http://localhost:3005/api/task

Esta URL nos presenta informacion sobre todas las tareas que estan asignadas
Todas las respuestas devolverán datos en formato json. Por cada Tarea se podrá visualizar :

  • tittle:"Se presenta el titulo de la tarea"
  • description:"Se presenta la descripcion de la tarea"
  • done:"Se presenta si la tarea ha sido realizada "

URL PARA PRESENTAR TAREAS POR TITULO:

http://localhost:3005/api/task?page=0&size=5&title=tarea numero uno

Este url nos presenta las tareas que tenga el titulo(En este caso el titulo asignado) y en que pagina esta ubicado
Todas las respuestas devolverán datos en formato json. Por cada Tarea se podrá visualizar :

  • tittle:"Se presenta el titulo especifico de la tarea"
  • description:"Se presenta la descripcion de la tarea"
  • done:"Se presenta si la tarea ha sido realizada "

URL PARA PRESENTAR POR NUMERO DE ITEMS:

http://localhost:3005/api/task?page=0&size=1

Este url nos presenta un numero de tareas que se especifique en el link mediante el size
Todas las respuestas devolverán datos en formato json. Se podran visualizar un numero de tareas :

URL PARA PRESENTAR MEDIANTE EL ID:

http://localhost:3005/api/task/64b0762605253069fcd94133

Este url nos presenta las tareas mediante el id unico que tienen
Todas las respuestas devolverán datos en formato json. Se podran visualizar las tareas mediante el id :

URL PARA PRESENTAR TAREAS QUE ESTEN VERIFICADAS:

http://localhost:3005/api/task/done

Este url nos presenta las tareas que tengan la propiedad "true"
Todas las respuestas devolverán datos en formato json. Se podran visualizar las tareas que esten validadas :
